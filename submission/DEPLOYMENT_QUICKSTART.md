# Deployment Quickstart (for Live Demo Link)

## Option A: Streamlit Community Cloud (Recommended)

1. Push this project to GitHub.
2. Go to share.streamlit.io and sign in.
3. Click New app and select your repository.
4. Set main file path to app.py.
5. Add secrets in app settings:
   - GROQ_API_KEY = your key
6. Deploy and copy the public app URL.

Use that URL as your Live Demo Link in submission.

## Option B: Record Demo Without Public Hosting

If public hosting is not possible before deadline:

- Keep repository public/private as required
- Record Loom while app runs locally
- Include clear working screenshots
- In notes, mention local run command used

This is less ideal than a public live link, but acceptable in some evaluation flows if clearly documented.

## Local Run Command (already working)

streamlit run app.py
